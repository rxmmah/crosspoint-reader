3. Dictionary Picker -- Scanner Validation

Place all test dictionaries in one of the following directories on the SD card: /.dictionaries/ or /dictionaries/ (checked in this order; only the first found is used). No word lookups are performed in this chapter. Open Settings -> Dictionary for every test below.

Test A: Folder without .ifo appears in picker

The no-ifo folder has a valid .idx and .dict but no .ifo file. It must appear in the picker list. Long-pressing it must show a "No metadata available" message.

Dictionary

no-ifo

Steps

  1. Open the dictionary picker.
  2. Confirm no-ifo is present in the list.
  3. Long-press no-ifo.
  4. Info view shows: No metadata available.
  5. Press Back. Picker reappears.

Test B: Folder with only .dict is hidden

The only-dict folder must not appear in the picker list.

Dictionary

only-dict

Steps

  1. Open the dictionary picker.
  2. Confirm only-dict is absent from the list.

Test C: Folder with multiple .ifo files is hidden

The multi-ifo folder must not appear in the picker list.

Dictionary

multi-ifo

Steps

  1. Open the dictionary picker.
  2. Confirm multi-ifo is absent from the list.

Test D: All test dictionaries appear in picker

All of the following must be visible in the picker. Do not select any yet.

Words

  • all-prep
  • basic
  • en-es
  • extract-dict
  • extract-syn-gen-syn
  • fail-decompress
  • full
  • gen-idx
  • gen-syn
  • long-prep
  • mini-prep
  • no-ifo
  • overflow-fields
  • phrase

Note

long-prep and mini-prep are for Ch 5 per-book prepare tests only. Do not select them from Settings.

Test E: Failed decompression -- error shown, prior selection unchanged

The preparation must fail with an error. The picker must remain open and the previously selected dictionary must be unchanged. No crash.

Dictionary

fail-decompress

Steps

  1. Note the currently selected dictionary before starting.
  2. Select fail-decompress in the picker.
  3. The preparation screen appears listing one step: Extract dictionary
  4. Press Confirm.
  5. Screen shows: Preparation failed
  6. Picker remains open. Previously selected dictionary is unchanged.

Test F: Metadata info view -- basic

Long-pressing a dictionary with a valid .ifo must open the parsed metadata info view showing at minimum: Name, Words, Alt Forms, and Type.

Dictionary

full

Steps

  1. Open the dictionary picker.
  2. Long-press full.
  3. Parsed info view appears. Verify Name, Words, Alt Forms, and Type are shown.
  4. Press Back. Picker reappears.

Test G: Metadata info view -- overflow fields

All displayed fields must wrap correctly when their values are very long. No truncation. No crash.

Dictionary

overflow-fields

Steps

  1. Open the dictionary picker.
  2. Long-press overflow-fields.
  3. Parsed info view appears.
  4. Verify Name, Description, Website, Date, Words, and Type are all shown and wrap correctly.
  5. Press Back. Picker reappears.

Test H: Picker pagination

With enough test dictionaries present, the list spans more than one screen. Hold-Up and Hold-Down must page-jump without crash.

Note

If the list fits on one screen, copy and rename any existing dictionary folder on the SD card to create additional entries until the list spans two screens.

Steps

  1. Open the dictionary picker.
  2. Hold Up. List jumps a page upward. No crash.
  3. Hold Down. List jumps a page downward. No crash.

Test I: Metadata info view -- View Raw

From the parsed metadata info view, pressing Confirm must switch to the raw .ifo text view showing the unprocessed file contents. Pressing Back from the raw view must return to the parsed view.

Dictionary

full

Steps

  1. Open the dictionary picker.
  2. Long-press full. Parsed info view appears.
  3. Press Confirm. Raw .ifo text view appears showing file contents.
  4. Press Back. Parsed info view reappears.
  5. Press Back. Picker reappears.

Test J: Compressed dictionary shows "Compressed" status line

The info view for a compressed dictionary must include a "Compressed" status line in addition to the standard metadata fields.

Dictionary

mini-prep (do not select it -- long-press only)

Steps

  1. Open the dictionary picker.
  2. Long-press mini-prep. Parsed info view appears.
  3. Verify a Compressed status line is shown.
  4. Press Back. Picker reappears.

Test K: "None found" when no dictionaries are present

When no valid dictionary folders exist on the SD card, the picker must show a "None found" message rather than an empty list.

Note

This test requires temporarily removing all dictionary folders from the SD card. Move them to a temporary folder on the card or on your computer, then run the test, then move them back before continuing.

Steps

  1. Move all dictionary folders off the SD card (or into a non-dictionary location).
  2. Open Settings -> Dictionary.
  3. Verify the picker shows: None found
  4. Press « Back.
  5. Restore all dictionary folders to their original location before continuing.
