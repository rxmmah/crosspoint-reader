15. Edge Cases

These tests cover back navigation, non-word selection, SD card state changes, and disabling the dictionary.

Dictionary

english-full

Test A: Back from definition returns to correct position

After viewing a definition and pressing « Back, the reader must return to exactly the same page position with no layout shift.

Steps

  1. Select any word from the list below.
  2. Definition screen appears.
  3. Press « Back.
  4. Verify you are returned to this page at the same position.
  5. Verify no layout shift has occurred.

Words

  • apple
  • ocean
  • shadow
  • flame
  • valley

Test B: Non-word tokens are filtered or produce no-match

Tokens consisting entirely of punctuation (no letters or digits) are filtered from the word list and cannot be navigated to. Tokens that contain digits or letters but have no dictionary match must produce a not-found result — no crash.

Steps

  1. Open word-select mode on this page.
  2. Attempt to navigate to a standalone punctuation token (period, hyphen). Verify: such tokens are absent from the word list — you cannot land on them with the navigation buttons.
  3. Navigate to the "3." list marker below (it contains a digit and remains selectable).
  4. Press Confirm.
  5. A not-found popup or brief "no word" popup appears. No crash. Word-select screen returns.

Items

  1. first
  2. second
  3. third

Test C: Dictionary removed mid-session

When the SD card is removed and reinserted without changing the dictionary setting, attempting a lookup must not crash the device. Acceptable outcomes are an error popup, the dictionary UI becoming hidden, or an automatic reset to None.

Steps

  1. With english-full selected and this book open, remove and reinsert the SD card, or power-cycle without changing the dictionary setting.
  2. Attempt to look up any word from the list below.
  3. Verify no crash occurs.

Words

  • apple
  • cloud
  • meadow

Test D: None selection disables dictionary UI

After selecting None, the dictionary word-select UI must no longer be accessible from the reader. Selecting english-no-syn prepares for Chapter 14.

Steps

  1. Navigate to Settings -> Dictionary. Select None.
  2. Return to the reader.
  3. Verify the dictionary word-select UI is no longer accessible.
  4. Return to Settings -> Dictionary. Select english-no-syn.
