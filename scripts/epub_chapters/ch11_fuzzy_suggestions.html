11. Fuzzy Suggestions

These are misspelled words. Because full has a .syn file, the alternate forms prompt appears first -- press Confirm each time. The words are not in .syn either, so the suggestions screen must appear next with nearby real headwords.

Dictionary

full

Test A: Suggestions list appears

Select a misspelled word. The alternate forms prompt appears -- press Confirm. No alternate form is found. The suggestions screen must appear with nearby headwords.

Steps

  1. Select applel.
  2. Alternate forms prompt appears. Press Confirm.
  3. Suggestions screen appears with nearby words.

Test B: Select a suggestion

From the suggestions screen reached in Test A, selecting a suggested word must open its definition screen.

Steps

  1. From suggestions screen, select any suggested word.
  2. Definition screen appears for that word.

Test C: Done button after selecting a suggestion

When arriving at a definition via the suggestions screen, the Confirm button must be labelled Done. Pressing Done must exit the definition screen and return to the reader.

Steps

  1. On the definition screen from Test B, verify Confirm is labelled Done.
  2. Press Done.
  3. Reader page returns. No crash.

Test D: Navigation across a long suggestions list

The word grafe is close in spelling to multiple headwords in full and must produce a suggestions list long enough to require scrolling. Both side buttons and front buttons must move the highlight one row at a time.

Steps

  1. Select grafe.
  2. Alternate forms prompt appears. Press Confirm.
  3. No alternate form found. Suggestions screen appears with 5 or more nearby headwords.
  4. Verify: side Up and Down buttons move the highlight one row at a time.
  5. Verify: front Left and Right buttons move the highlight one row at a time.
  6. Select any suggestion. Definition screen appears. No crash.
