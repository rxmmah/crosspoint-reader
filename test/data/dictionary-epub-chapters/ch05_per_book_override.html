5. Per-Book Dictionary Override

The reader menu includes a Book Dictionary option that lets you set a dictionary for the current book independently of the global setting in Settings -> Dictionary. The per-book choice is saved and restored each time the book is opened. Selecting Use Global clears the override and reverts to the global setting.

Test A: Set per-book dictionary from reader menu

This test verifies that selecting a dictionary via the book menu overrides the global setting and enables word lookup for this book.

Dictionary

prep-gen-idx (already prepared in Ch 3)

Steps

  1. Open the reader menu (press Confirm).
  2. Select Book Dictionary.
  3. The dictionary picker opens. Use Global is highlighted (no per-book override set yet).
  4. Select prep-gen-idx. The picker closes.
  5. Open the reader menu again. Verify Look Up Word is now visible.

Words

  • apple
  • bridge
  • cloud

Test B: Per-book override persists after closing and reopening

Closing the book and reopening it must restore the per-book dictionary setting automatically.

Steps

  1. Press Back (long press) to return to the home screen.
  2. Reopen this book.
  3. Open the reader menu. Verify Look Up Word is still visible.
  4. Select Book Dictionary. Verify prep-gen-idx is marked as selected.

Test C: Clear per-book override with Use Global

Selecting Use Global must clear the per-book override and revert lookup availability to whatever the global setting provides.

Steps

  1. Open the reader menu -> Book Dictionary.
  2. Select Use Global. The picker closes.
  3. Open the reader menu. Look Up Word visibility now reflects the global dictionary setting.

Note

If no global dictionary is set in Settings -> Dictionary, Look Up Word will disappear from the menu after selecting Use Global.

Test D: Per-book prepare -- prep-mini (all four steps, quick)

Selecting an unprepared dictionary from the book menu must trigger the same prepare workflow as selecting from Settings. All four steps must complete and the book dictionary must be set to prep-mini.

Dictionary

prep-mini (must not be pre-processed -- verify no .idx.oft or .syn.oft files exist)

Steps

  1. Open the reader menu -> Book Dictionary.
  2. Select prep-mini.
  3. Confirmation screen lists in order:
    • Extract dictionary
    • Extract alt forms
    • Generate dictionary offset file
    • Generate alt-form offset file
  4. Press Confirm.
  5. All four steps complete with [OK]. Preparation complete.
  6. Picker closes. Open the reader menu. Look Up Word is visible.
  7. Verify the Book Dictionary row shows prep-mini on the right side.

Words

  • apple
  • cloud
  • zenith

Test E: Per-book cancel prepare -- prep-long

Pressing Back during preparation must cancel without changing the current per-book dictionary. The picker must return with the previous selection still shown as active.

Dictionary

prep-long (must not be pre-processed -- verify no .idx.oft or .syn.oft files exist)

Note

The current per-book dictionary at the start of this test is prep-mini (set in Test D). Do not change it before starting.

Steps

  1. Open the reader menu -> Book Dictionary.
  2. Select prep-long.
  3. Confirmation screen appears listing four steps. Press Confirm to start.
  4. Processing begins. Press Back to cancel during any step.
  5. Screen shows: User cancelled
  6. The dictionary picker returns. prep-mini is still shown as selected.
  7. Press Back to close the picker without changing the book dictionary.

Test F: Global dictionary restored when book is closed

The per-book dictionary override must not affect the global dictionary setting. Closing the book must restore the global dictionary to whatever it was before the book was opened.

Note

The per-book dictionary for this book is currently prep-mini (set in Test D). The global dictionary in Settings is whatever was set before starting this chapter.

Steps

  1. Open Settings -> Dictionary. Note which dictionary is currently selected as the global setting.
  2. Return to this book. The per-book dictionary (prep-mini) is active.
  3. Press Back (long press) to close the book and return to the home screen.
  4. Open Settings -> Dictionary.
  5. Verify the global dictionary is unchanged from step 1.
