2. Document Conventions

Formatting

Italic text is verbatim on-screen text -- the exact words shown on the device.

  • Select english-full in the picker. -- english-full is the dictionary name as shown
  • Press Confirm. -- Confirm is the button label as shown
  • Screen shows: Preparation complete -- Preparation complete is the status message as shown

List types

  • Numbered list -- ordered steps. Follow them in sequence.
  • Bulleted list -- unordered items. Try them in any order.

Test section structure

Each test section may contain the following sub-headings, in this order. Sub-headings are omitted when they do not apply.

DictionaryWhich dictionary to have selected for this test
NoteSpecial preconditions or warnings
StepsOrdered actions to perform
WordsItems to look up or select
AdditionalExtra items for extended verification

Processing screen

Each preparation step shows a status prefix:

[ > ]Step currently running (bold text)
[OK]Step completed successfully
[!!]Step failed (bold text)
[   ]Step not yet started

After all steps finish, the screen shows either Preparation complete or Preparation failed.

Expected values

Some entries in a word list include the definition text to verify.

  • fire -- expect: fuego (m) / incendio (m)

The text after expect: must match the definition screen exactly.

Word select navigation

When the word select screen is active, the following button model applies. Tests in later chapters navigate to specific words without re-explaining these mechanics.

Up / DownMove between rows on the page
Left / RightMove between words within the current row
ConfirmSelect the highlighted word and begin lookup
BackExit word select and return to the reader without performing a lookup

This section is informational. Navigation is exercised implicitly by the lookup tests in later chapters and is not re-tested explicitly.

Definition screen navigation

When a definition screen is active, the following button model applies.

PageForward / RightGo to the next page of the definition (view mode only)
PageBack / LeftGo to the previous page of the definition (view mode only)
Confirm (Look Up Word)Enter word-select mode on the definition text
Back (short press)Return to caller (previous definition or word-select screen)
Back (long press)Done — exit all the way back to the reader

Page navigation is locked while word-select mode is active. This section is informational; these controls are exercised by the lookup tests in later chapters.

Draft chapters

Chapters with [Draft] in the title document features whose firmware code does not yet exist. Those tests cannot be performed.
