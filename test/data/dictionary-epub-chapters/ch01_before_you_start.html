1. Before You Start

Before running any test in this document, clear the book's render cache. A stale render cache can cause test steps to show outdated content, leading to false failures.

How to clear the cache

Use either method -- both work:

  1. SD card: delete the book cache folder from the .crosspoint/ folder on the SD card.
  2. Reader menu: open the test epub on the device, open the reader menu, and use the Clear book cache option.

This only needs to be done once after flashing a new build, not before every chapter.
