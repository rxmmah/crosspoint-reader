12. Lookup from Definition

These tests verify that the definition screen supports word-select mode. Pressing Look Up Word from a definition screen enters word-select mode directly on the definition text, allowing the user to look up any word in the definition without returning to the reader.

Dictionary

english-full

Test A: Look Up Word button label

After a successful lookup, the Confirm button on the definition screen must be labelled Look Up Word.

Steps

  1. Select any word from the reader. Definition screen appears.
  2. Verify the Confirm button is labelled Look Up Word.

Test B: Enter word-select mode on definition

Pressing Look Up Word from the definition screen must enter word-select mode on the definition text. A word highlight must appear and navigation must work.

Steps

  1. Look up apple. Definition screen appears.
  2. Press Look Up Word (Confirm). A word in the definition text becomes highlighted.
  3. Verify the initially highlighted word is approximately centred in the definition text — it must not be the very first word on the first line.
  4. Press Up / Down / Left / Right. Highlight moves to adjacent rows and words.
  5. Verify page navigation buttons do not change the page while in word-select mode.

Test C: Look up a word from a definition

Selecting a word in word-select mode must perform a lookup and open a new definition screen for that word.

Steps

  1. Look up chronicle. Definition screen appears.
  2. Press Look Up Word. Word-select mode activates on the definition text.
  3. Navigate to any content word in the definition.
  4. Press Confirm. Looking up… popup appears briefly.
  5. A new definition screen appears for the selected word. No crash.

Test D: Chained lookup

From a definition reached via Look Up Word, pressing Look Up Word again must open a third definition screen. No crash at any depth.

Steps

  1. Look up chronicle. Definition screen 1 appears.
  2. Press Look Up Word. Enter word-select mode.
  3. Select a word. Definition screen 2 appears.
  4. Press Look Up Word again. Enter word-select mode on definition 2.
  5. Select a word. Definition screen 3 appears. No crash.

Test E: Short Back exits word-select mode

While in word-select mode, a short press of Back must exit word-select and return to the definition view. It must not exit to the reader or to the EPUB word-select screen.

Steps

  1. Look up any word. Definition screen appears.
  2. Press Look Up Word. Word-select mode activates.
  3. Short-press Back.
  4. Word-select mode exits. Definition view is shown again. No lookup performed.

Test F: Long-press Back exits to reader

A long press of Back (hold until released) from any level of the definition screen must exit all the way back to the reader page.

Steps

  1. Look up any word. Definition screen appears (view mode).
  2. Long-press Back. Reader page returns immediately. No crash.
  3. Look up any word. Press Look Up Word. Word-select mode activates.
  4. Long-press Back. Reader page returns immediately. No crash.

Test G: Page lock during word-select

While in word-select mode, pressing page navigation buttons must not change the page. Page navigation must resume normally after exiting word-select mode.

Steps

  1. Look up chronicle (multi-page definition). Advance to page 2.
  2. Press Look Up Word. Word-select mode activates on page 2.
  3. Press PageForward and PageBack. Page must not change.
  4. Short-press Back. Word-select mode exits.
  5. Press PageBack. Page 1 appears normally.
