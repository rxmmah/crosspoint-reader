6. Direct Lookup

These tests cover the basic lookup path: a headword found directly in the index returns a definition immediately. No stemming, alternate forms prompt, or suggestions screen is involved.

Dictionary

english-full

Test A: Definition returned immediately

Each word below is a headword in english-full. Selecting it must open the definition screen directly with no alternate forms prompt or suggestions screen.

Words

  • cloud

Test B: Multi-page definition navigation

The word chronicle has a long definition that spans more than one page. The page navigation controls must become active and both forward and backward navigation must work without a crash.

Steps

  1. Select chronicle. Definition screen appears on page 1.
  2. Press PageForward or Right. Page 2 appears.
  3. Press PageBack or Left. Page 1 reappears.
  4. Long-press Back. Reader returns.

Test C: Definition viewer honors reader font size

The definition viewer must use the same font size as the reader. Changing the font size in Settings must be reflected in the definition screen.

Steps

  1. Note the current font size in Settings → Reader → Font Size.
  2. Look up any word from Test A. Note the text size in the definition screen.
  3. Long-press Back to return to the reader.
  4. Go to Settings → Reader → Font Size. Change to a different size.
  5. Return to this chapter. Look up the same word.
  6. Verify the definition screen text size has changed to match the new setting.
  7. Restore the original font size when done.

Test D: Look Up Word button on direct hit

After any successful lookup from Test A, the Confirm button on the definition screen must be labelled Look Up Word. This confirms showLookupButton is set correctly for direct hits.

Steps

  1. Look up any word from Test A.
  2. Verify Confirm button label is Look Up Word.

Test E: Initial cursor position in reader word-select

When entering word-select mode from the reader, the initially highlighted word must be approximately centred on the page — not the first word on the first line.

Steps

  1. Open word-select mode on this page (press the word-select button from the reader).
  2. Verify the highlighted word is somewhere in the middle of the visible text, not at the top-left corner.
  3. Press Back to exit word-select without looking anything up.

Test F: Word-select overlay alignment with definition font override

When the definition font or size differs from the reader font, word-select highlights in the definition viewer must still align accurately over the rendered text.

Steps

  1. Go to Settings → Reader → Definition Font Size. Change to a size different from the reader font size.
  2. Return to this chapter. Look up chronicle.
  3. Press Look Up Word to enter word-select mode on the definition.
  4. Navigate to several words. Verify each highlight box is positioned directly over the word it represents — no offset or misalignment.
  5. Long-press Back to return to the reader.
  6. Restore Settings → Reader → Definition Font Size to its original value.
