13. Phrase / Multi-word Lookup

These tests verify multi-word phrase selection in both the EPUB reader and the definition viewer. In word-select mode, a long press of the Confirm button enters multi-word select mode. The word that was highlighted when you long-pressed becomes the anchor — it stays fixed. Use the navigation buttons to move the cursor to extend or shrink the selection. A short press of Confirm then looks up the selected phrase. Back exits multi-word select and returns to single-word select mode.

Dictionary

phrase (switch to the Phrase Sampler dictionary before starting)

Test phrases

Use the words below as lookup targets throughout this chapter.

Two words: for you — pork scratching

Five words: have a whale of a time — have a good time

Eight words: march to the beat of a different drum

Test A: Enter multi-word select from the reader

A long press of Confirm while in single-word select mode must activate multi-word select. The anchor and cursor start on the same word, so initially only one word is highlighted.

Steps

  1. Enter word-select mode (select any word from the reader).
  2. Long-press Confirm.
  3. Verify that word-select mode is still active and exactly one word is highlighted (the anchor). No crash, no accidental lookup.

Test B: 2-word phrase lookup from reader

Selecting a 2-word phrase must look it up successfully.

Steps

  1. In word-select mode, navigate to for in the phrase "for you".
  2. Long-press Confirm to enter multi-word select. for is the anchor.
  3. Press Right (or Down then Left as needed) to move the cursor to you. Both words are now highlighted.
  4. Short-press Confirm. The definition for for you appears.

Test C: 5-word phrase lookup from reader

Selecting a longer phrase across multiple navigation moves must work correctly.

Steps

  1. In word-select mode, navigate to have in the phrase "have a whale of a time".
  2. Long-press Confirm. have is the anchor.
  3. Press Right four times to extend the cursor to time. All five words are highlighted.
  4. Short-press Confirm. The definition for have a whale of a time appears.

Test D: Back exits multi-word select

Pressing Back while in multi-word select mode must return to single-word select mode, not exit word-select entirely.

Steps

  1. Enter word-select mode. Long-press Confirm to enter multi-word select.
  2. Move the cursor to extend the selection over two or more words.
  3. Press Back.
  4. Verify: back in single-word select mode — only the anchor word remains highlighted. Navigation still works. No exit to reader.
  5. Press Back again (short press). Word-select exits, returning to the reader.

Test E: Multi-word select from the definition viewer

Multi-word select must also work when in word-select mode on a definition screen. This test chains two lookups: a single-word select brings up a definition, then multi-word select picks a phrase from within that definition.

Steps

  1. In word-select mode, navigate to have in the phrase "have a good time" above.
  2. Long-press Confirm and extend the cursor to time (5 words: have a good time).
  3. Short-press Confirm. The definition for have a good time appears. Verify the definition text reads: "…Compare: have a whale of a time."
  4. Press Confirm (Look Up Word) to enter word-select mode on the definition.
  5. Navigate to have in the phrase "have a whale of a time" in the definition text.
  6. Long-press Confirm. Enter multi-word select. Extend cursor to time (5 words).
  7. Short-press Confirm. The definition for have a whale of a time appears. No crash.

Known Limitation: Page boundaries

Multi-word selection cannot span a page boundary. Words on the next page are not loaded into memory, so extending the selection past the last word on the current page is not possible.

Workaround: Reduce the reader font size in Settings -> Reader -> Font Size so that more words fit on a single page. Perform the phrase lookup, then restore the original font size.

A complete fix would require significant changes to the rendering engine and is out of scope for the dictionary feature.

Test F: Phrase not in dictionary

Selecting a word sequence that does not appear in the dictionary must trigger the normal not-found flow (popup or suggestions), not a crash.

Steps

  1. Enter multi-word select and select any 3-word sequence not listed above (e.g. three consecutive common words).
  2. Short-press Confirm.
  3. Verify: not-found popup appears, or suggestions screen appears if close matches exist. No crash. Navigation returns normally after dismissal.
